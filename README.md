## 2. 주요기능

- 근태관리솔루션과 연동한 자동제어시스템
- 원격제어 기능
- 조밀한 지역제어 가능
- 구역별, 직원별 실시간 전력모니터링
- 월별 전력 사용량 집계 확인
- ESS와 연동 가능
